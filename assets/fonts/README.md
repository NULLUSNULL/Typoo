# Tipografías empaquetadas

Estas tipografías se distribuyen con Typoo y se cargan al arrancar la
aplicación (`core/fuentes.py`). Todas están bajo la **SIL Open Font License,
Version 1.1**. El texto íntegro de cada licencia está en `licenses/`.

| Familia | Diseñador / Proyecto | Fuente |
|---------|----------------------|--------|
| EB Garamond | Georg Duffner, Octavio Pardo | Google Fonts |
| Lora | Cyreal | Google Fonts |
| Literata | TypeTogether | Google Fonts |
| Crimson Pro | Jacques Le Bailly, Sebastian Kosch | Google Fonts |
| Spectral | Production Type | Google Fonts |
| Bitter | Sol Matas (Huerta Tipográfica) | Google Fonts |
| Playfair Display | Claus Eggers Sørensen | Google Fonts |
| Inter | Rasmus Andersson | Google Fonts |

La SIL OFL permite usar, modificar y redistribuir estas fuentes, incluso de
forma comercial, siempre que se mantengan los avisos de licencia.
